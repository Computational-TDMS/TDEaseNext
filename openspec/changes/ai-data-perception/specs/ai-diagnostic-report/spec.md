## ADDED Requirements

### Requirement: Empathic Data Diagnosis
The AI Agent SHALL read the Data Hologram and generate an "Empathic Diagnostic Report"—a natural language, scientifically nuanced narrative explaining the inner logic and health of the user's data.

#### Scenario: Providing data health storytelling
- **WHEN** the user opens the diagnostic panel for a completed sample
- **THEN** the AI responds not just with numbers, but with expert intuition: "Your sample exhibits excellent signal-to-noise in the 800-1200 m/z range, but I sense a chaotic fragmentation pattern near the C-terminus, suggesting potential over-digestion or novel cleavage events."

### Requirement: Conversational Iteration
The user SHALL be able to debate the diagnostic report with the Agent, probing specific data anomalies.

#### Scenario: Debating a peak anomaly
- **WHEN** the user asks, "Why do you think the 1500 m/z region is chaotic?"
- **THEN** the agent references the Data Hologram's sub-features and explains its reasoning based on the isotopic spacing irregularity.
